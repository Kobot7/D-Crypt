Vernam Cipher


How it Works

Vernam cipher is also known as a "One Time Pad". It makes use of a random set of non-repeating input characters that is only used once. This makes it highly secure as the pad is never reused for encryption. The length of the one-time pad is equal to the length of the plaintext.

It has been classified as both a substitution technique as well as a transposition technique.

Vernam cipher encrypts plaintext using the following steps:

  1. Encode each plaintext letter as a number in an increasing sequence. i.e. A = 0, B = 1, C = 2, ... Z = 25
  2. Encode each letter of the one-time pad in the same way.
  3. Add each number corresponding to the plaintext letter to the corresponding one time pas number.
  4. Calculate modulo 26 of the sum.
  5. Translate each number of the sum back to the corresponding alphabet. This produces the ciphertext.
