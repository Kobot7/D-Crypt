Simple Columnar Transposition Technique


How it Works

Simple columnar transposition technique encrypts plaintext using the following steps:

  1. Write the plaintext row-by-row. The number of columns is determined by the length of the key.
  2. Read the output column-by-column, in the order of the key.
  3. The read message is the ciphertext.
